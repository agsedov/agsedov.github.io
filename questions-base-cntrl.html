Дополнительная контрольная (1 семестр ИТ)

Если ваша сумма баллов к концу семестра составляет < 10 баллов, то вы должны её написать. Контрольная проводится в конце декабря(после четвёртой). Будут даны 2 задачи из списка:

  1. Решить систему линейных уравнений методом Гаусса.
    В качестве подготовки можно прорешать номер 8.1 из задачника Кострикина.
  2. Найти все корни n-ой степени из комплексного числа.
    В качестве подготовки можно прорешать номер 22.7 из задачника Кострикина.
  3. Вычислить ранг матрицы методом окаймляющих миноров
    В качестве подготовки можно прорешать номер 7.1 из задачника Кострикина.
  4. Решить систему методом Крамера
    В качестве подготовки можно прорешать номер 8.6 из задачника Кострикина.
  5. Решить матричное уравнение вида AXB=C
    В качестве подготовки можно прорешать номер 18.3 из задачника Кострикина.
  6. Определить, является ли множество с указанной бинарной операцией группой.
    В качестве подготовки можно прорешать номер 55.1 из задачника Кострикина.
  7. Найти проекцию вектора a на плоскость b,c параллельно вектору d
    В качестве подготовки можно решить номер 31 из задачника Моденова и Пархоменко.
  8. Найти матрицу перехода P от базиса S к базису S'.
    В качестве подготовки можно решить номер 34.11 из задачника Кострикина (без доказательства).
  9. Найти объём параллелепипеда построенного на векторах a,b,c
    В качестве подготовки можно решить номер 190 из задачника Моденова и Пархоменко.
  10. Найти НОД двух многочленов
    В качестве подготовки можно прорешать номер 25.2 из задачника Кострикина.
  11. Найти рациональные корни многочлена
    В качестве подготовки можно прорешать номер 28.2 из задачника Кострикина.
  12. Разложить многочлен на неприводимые множители над указанным полем (ℝ или ℂ)
    В качестве подготовки можно прорешать номер 27.1а)б)в)г) из задачника Кострикина.

    13. На решение задач даётся 1 час. Нужно решить обе задачи.
